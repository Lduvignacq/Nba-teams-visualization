# Nba-teams-visualization
Visualizing the performance of nba teams throughout the years
# Nba_api
nba met à dispo son api pour creer des datasets bcp plus carré, on peut l'utiliser avec pip install nba_api. 
L'avantage, c'est qu'il y'a pleins de projets sur internet qui l'utilisent, donc on a déja bcp d'exemple.
# fichier
data contient toutes les données.
mainpage.html est la page principale,
team-map.js se charge de produire la page.
nba.js contient la passinge wheel, et la hotmap pour le moment.
